PoC RADIO LAB EUROPE — MULTILINGUAL VERSION
================================================

LANGUAGES
- EN — English
- PL — Polski
- DE — Deutsch

HOW TO CHECK LOCALLY
1. Unzip the package.
2. Open index.html.
3. Use EN / PL / DE in the upper navigation.
4. The selected language is remembered by the browser.

HOW TO PUBLISH
Upload all files and folders to the root of the GitHub repository,
replacing the previous website files.

IMPORTANT
Keep:
- CNAME
- assets/
- reports/
- robots.txt
- sitemap.xml

The approved radio hero image is already included.
